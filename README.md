# Управление кешем в API Shop

## Подключить
### Composer
```json
{
  "require": {
    "pllano/cache": "~1.0.0"
  }
}
```
### AutoRequire
```json
{
    "require": [{
        "namespace": "Pllano\\Caching",
        "dir": "/pllano/cache/src",
        "link": "https://github.com/pllano/cache/archive/master.zip",
        "git": "https://github.com/pllano/cache",
        "name": "cache",
        "version": "master",
        "vendor": "pllano",
        "state": "1",
        "system_package": "1"
    }]
}
```
